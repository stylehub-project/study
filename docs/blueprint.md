# **App Name**: Sunrise Study Club

## Core Features:

- User Authentication and Role Management: Secure user signup and login with role-based access control (student, teacher, club member, admin) and managed in Firestore.
- Dashboard: Landing page with student name, status tiles (attendance, today's class, next assignment), course progress, quick tasks, weekly focus, and curated tips.
- Assignments & Submissions: Functional upload panel for video/audio homework submission with controlled uploader.
- Sunrise Club Zone: Exclusive zone for club members featuring weekly practice uploads.
- Profile Setup: Mandatory initial profile setup with skippable stepper modal to capture user details (name, email, role, class/grade, club ID).
- Content Recommendations: AI-powered tool to provide personalized study materials recommendations for students based on enrolled courses.
- Member Roster: Display an enablable skeleton list of Club members.

## Style Guidelines:

- Background: Deep navy/ink blue (#191970) for dashboards to promote calm and professionalism.
- Panel: Warm paper-like hue (#F5F5DC) for study content to enhance readability.
- Accent: Sunrise-gold (#FFB347) for interactive elements and highlights.
- Secondary: Teal (#00C2A8) to provide visual interest without being distracting.
- Headline font: 'Manrope' sans-serif for a friendly and polished appearance.
- Body font: 'System UI' (sans-serif) to ensure optimum readability for long study sessions.
- Font sizes: H1 (28-36px), H2 (22-28px), body (14-16px).
- Simple, clean icons for navigation and key actions; icons should be in sunrise-gold.
- Mobile-first, responsive design with breakpoints at sm (640px), md (768px), lg (1024px), xl (1280px).
- On desktop, use a centered max-width container (1400px) with left rail navigation.
- Subtle, slow (200-350ms) easing for transitions; avoid distracting motion during study flow.
